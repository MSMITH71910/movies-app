About This Website

Hi, I'm Michael R Smith! Welcome to my movie collection website.

This website was built using Python and the Django web framework. Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It provides a robust set of tools and features that make it easy to build web applications efficiently.

The purpose of this site is to serve as a personal project that showcases my favorite movies. It allows users to add, edit, and delete movies, as well as view details such as the title, description, and release date. This project not only demonstrates my skills in web development but also serves as a portfolio piece as I pursue a career in software development.

Through this application, I aim to highlight my understanding of web technologies, including HTML, CSS, and JavaScript, along with my proficiency in Python and Django. I hope to continue enhancing this project with additional features and improvements as I grow in my career.
